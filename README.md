## Django, python, Mysql을 동시에 연동하는 프로젝트 입니다.

### 소개 페이지
    - 사용된 스택과 자기소개, 활동 내역 페이지 입니다.
### 연동 페이지
    - mysql과 연동하여 가장 기본적인 데이터베이스를 보여주는 페이지 입니다.
### 분석 페이지
    - 연동 페이지를 참조해 python으로 분석한 정보를 보여주는 페이지 입니다.
### 시각화 페이지
    - 연동 페이지에 출력된 자료를 chart.js를 이용하여 여러 그래프로
    - 시각화 하였습니다.
